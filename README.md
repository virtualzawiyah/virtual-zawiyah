# Virtual Zawiyah

A digital space for Islamic learning, reflection, and community engagement.

## Getting Started

First, install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
